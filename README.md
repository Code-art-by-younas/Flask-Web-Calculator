# 🧮 Flask Web Calculator

This is a simple web-based calculator built using Python Flask.

## 🚀 Features
- Add
- Subtract
- Multiply
- Divide
- Simple web UI

## 🛠️ Tech Stack
- Python
- Flask
- HTML
- CSS

## ▶️ How to Run
1. Install Flask:
pip install flask



2. Run app:
python app.py



3. Open browser:
http://127.0.0.1:5000/
